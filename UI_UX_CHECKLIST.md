## UI/UX чеклист (TC‑47)

### Браузеры
- Chrome (desktop)
- Edge (desktop)
- Firefox (desktop)

### Разрешения/адаптивность
- 1920x1080
- 1366x768
- 390x844 (мобильный режим) — проверяем, что основные страницы читаемы

### Ключевые страницы
- Вход
- Заказы: список/создание/карточка
- Клиенты: список/карточка
- Склад: список/операции
- Отчёты: выгрузка XLSX
- Администрирование: настройки/бэкап

### Что проверяем
- Нет “битых” кнопок/пустых страниц
- Ошибки API показываются пользователю сообщением
- Важные действия доступны за 2–3 клика (ТЗ 5)
